iSee P2P

iSee P2P enables two-party peer-to-peer video-conferencing calls for public IP addresses.

 iSee P2P features
  • Chat 
  • Two-party peer-peer video-conferencing calls 
  • Simple and easy to use interface
  • Small size for fast downloads
  • Free Directory Service: enables users to find the dynamic IP address of each other
 iSee P2P system requirements
  • OS - MS Windows 9X, NT, XP
